dev-box
=======

Dev box using vagrant

Prerequisite
-------------
Virtual box: 4.2.8 (https://www.virtualbox.org/wiki/Download_Old_Builds)  
Vagrant: 1.2.2 (http://downloads.vagrantup.com/)

First time
-----------
git clone git@github.com:vinodpandey/dev-box.git
vagrant up

Suspend/Resume VM
-----------------
vagrant suspend
vagrant resume

Shutdown/destroy
----------------
vagrant halt
vagrant destroy
